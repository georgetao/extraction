Our Models Technical Approach

ExtrAction is a tool that summarizes relevant action items in emails and provides them to the user in an easy-to-use format. It is built on two different natural language processing models. The first, a classification model, classfies each sentence in the email as containing an action item or not. The second model summarizes each action sentence into a to-do list item.
